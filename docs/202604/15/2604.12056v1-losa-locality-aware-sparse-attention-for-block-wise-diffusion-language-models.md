## 摘要
分块式扩散语言模型 (DLM) 以任意顺序生成多个 token，为自回归解码流水线提供了一种极具前景的替代方案。然而，在长上下文场景下，它们仍受限于受内存限制的注意力机制瓶颈。由于 KV 膨胀 (KV Inflation) 问题，朴素的稀疏注意力在 DLM 上失效，即不同的查询选择不同的前缀位置，导致访问的 KV 页并集过大。为了解决这一问题，我们观察到在连续的去噪步骤之间，只有一小部分活跃 token 表现出显著的隐藏状态变化，而大多数稳定 token 几乎保持不变。基于这一洞察，我们提出了 LOSA（局部感知稀疏注意力），它为稳定 token 复用缓存的前缀注意力结果，并仅对活跃 token 应用稀疏注意力。这大幅减少了必须加载的 KV 索引数量，从而同时实现了更高的加速比和更高的准确率。在多个分块式 DLM 和基准测试中，LOSA 在显著提高效率的同时保持了接近密集注意力的准确率，在极高稀疏度水平下，平均准确率提升高达 9 个百分点，同时保持了 1.54 倍更低的注意力密度。它还在 RTX A6000 GPU 上实现了高达 4.14 倍的注意力加速，证明了所提方法的有效性。

## Abstract
Block-wise diffusion language models (DLMs) generate multiple tokens in any order, offering a promising alternative to the autoregressive decoding pipeline. However, they still remain bottlenecked by memory-bound attention in long-context scenarios. Naive sparse attention fails on DLMs due to a KV Inflation problem, where different queries select different prefix positions, making the union of accessed KV pages large. To address this, we observe that between consecutive denoising steps, only a small fraction of active tokens exhibit significant hidden-state changes, while the majority of stable tokens remain nearly constant. Based on this insight, we propose LOSA (Locality-aware Sparse Attention), which reuses cached prefix-attention results for stable tokens and applies sparse attention only to active tokens. This substantially shrinks the number of KV indices that must be loaded, yielding both higher speedup and higher accuracy. Across multiple block-wise DLMs and benchmarks, LOSA preserves near-dense accuracy while significantly improving efficiency, achieving up to +9 points in average accuracy at aggressive sparsity levels while maintaining 1.54x lower attention density. It also achieves up to 4.14x attention speedup on RTX A6000 GPUs, demonstrating the effectiveness of the proposed method.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：分块式扩散语言模型（Block-wise DLMs）在处理长文本时面临严重的**内存受限（Memory-bound）**瓶颈。
*   **研究背景**：与自回归模型不同，DLM 在每个去噪步骤中会同时更新一个块（Block）内的多个 token。虽然可以应用传统的稀疏注意力（如 QUEST）来加速，但 DLM 存在**“KV 膨胀”（KV Inflation）**问题：块内不同的查询（Query）会选择不同的前缀 KV 位置，导致最终需要加载的 KV 缓存并集（Union）非常大，抵消了稀疏化带来的加速效果。
*   **整体含义**：本文旨在通过利用扩散模型去噪过程中的“表示变化局部性”，减少 KV 缓存的加载量，从而在不损失精度的情况下显著提升长文本 DLM 的推理速度。

### 2. 论文提出的方法论：LoSA
*   **核心思想**：观察到在连续的去噪步骤中，只有少数 token（**活跃 token**）的隐藏状态发生显著变化，而大多数 token（**稳定 token**）的表示几乎保持不变。
*   **关键技术细节**：
    1.  **局部性剪枝（Locality Pruning）**：通过计算当前步与前一步查询向量（Q）之间的均方误差（MSE）来量化每个 token 的变化程度。
    2.  **活跃/稳定分类**：将 MSE 最高的 Top-k 个 token 标记为“活跃”，其余为“稳定”。
    3.  **缓存复用**：对于“稳定 token”，直接复用前一步缓存的前缀注意力输出（$o_p$）和归一化因子（$L_p$）。
    4.  **局部感知稀疏注意力**：仅针对“活跃 token”运行稀疏选择算法（如 QUEST），计算这些活跃 token 所需 KV 索引的并集。由于参与计算的查询数量大幅减少，KV 并集的大小显著缩小。
    5.  **在线 Softmax 合并**：使用在线 Softmax 技术将复用的/新计算的前缀注意力结果与块内双向注意力结果进行合并。

### 3. 实验设计

*   **充分性评价**：实验设计较为充分。不仅对比了准确率，还详细分析了 KV 缓存密度、端到端延迟分解以及不同块大小（Block Size）的影响。对比实验涵盖了最新的同类竞争方法，结果具有较强的说服力和客观性。

### 6 主要结论与发现
*   **性能提升**：在极高稀疏度下（预算 128），LoSA 比 QUEST 在 LongBench 上的平均准确率高出 **9.01 个百分点**。
*   **效率优势**：LoSA 的注意力密度比 QUEST 低 **1.54 倍**，在 RTX A6000 上实现了高达 **4.14 倍** 的注意力加速。
*   **精度保持**：LoSA 能够保持接近密集注意力的精度，因为它为稳定 token 保留了完整的历史注意力信息，而非像传统稀疏方法那样进行截断。

### 7. 优点
*   **洞察深刻**：敏锐捕捉到了 DLM 去噪过程中表示变化的局部性特征，并将其转化为计算优势。
*   **解决痛点**：有效解决了 DLM 推理中特有的“KV 膨胀”问题。
*   **即插即用**：作为一种推理侧的优化方案，不需要对预训练模型进行微调或架构修改。
*   **软硬件结合**：提供了高效的 Triton/CUDA 实现，证明了算法在实际硬件上的落地能力。

### 8. 不足与局限
*   **短文本收益有限**：在短文本（<1K token）场景下，KV 膨胀问题不明显，LoSA 的优势相对减弱。
*   **初始开销**：每个块的第一步去噪需要一次全量密集注意力计算来初始化缓存，这带来了一定的固定开销。
*   **批处理限制**：目前的评估主要集中在 Batch Size = 1 的场景，尚未深入探讨在复杂并发服务系统（如 vLLM）中的集成表现。
